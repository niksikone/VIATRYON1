# VIATRYON Color Scheme

## White Background Theme with Premium Green

### Color Palette

#### Primary Colors
- **Premium Green**: `emerald-500` (#10B981) - Main brand color
- **Dark Green**: `emerald-600` (#059669) - Hover states
- **Light Green**: `emerald-400` (#34D399) - Accents
- **Extra Light Green**: `emerald-100` (#D1FAE5) - Backgrounds
- **Pale Green**: `emerald-50` (#ECFDF5) - Subtle backgrounds

#### Neutral Colors
- **White**: `white` (#FFFFFF) - Main background
- **Light Grey**: `gray-50` (#F9FAFB) - Section backgrounds
- **Border Grey**: `gray-200` (#E5E7EB) - Borders
- **Text Grey**: `gray-600` (#4B5563) - Secondary text
- **Dark Grey**: `gray-900` (#111827) - Primary text

### Usage Guidelines

#### Backgrounds
- Main background: `bg-white`
- Alternate sections: `bg-gray-50` or `bg-emerald-50`
- Cards: `bg-white` with `border-gray-200`

#### Text
- Headlines: `text-gray-900`
- Body text: `text-gray-600`
- Links: `text-emerald-600` hover `text-emerald-700`

#### Buttons
- Primary CTA: `bg-emerald-500 text-white` hover `bg-emerald-600`
- Secondary CTA: `border-emerald-500/50 text-emerald-400` hover `bg-emerald-500/10`

#### Accents
- Icons backgrounds: `bg-emerald-100`
- Badges: `bg-emerald-50 border-emerald-200 text-emerald-700`
- Numbers/Steps: `text-emerald-500`
